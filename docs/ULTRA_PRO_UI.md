# Bản Ultra Pro UI

Bản này nâng cấp mạnh phần web bán hàng, giữ nguyên Admin gốc.

## Nâng cấp chính

- Header chuyên nghiệp, thanh tìm kiếm lớn, menu danh mục.
- Hero lớn kiểu siêu thị linh kiện máy tính.
- Quảng cáo 2 bên, flash sale, quick tabs danh mục.
- Bộ lọc sản phẩm theo từ khóa, danh mục, khoảng giá, sắp xếp.
- Card sản phẩm cao cấp, nút chi tiết, giỏ hàng, so sánh.
- Build PC theo ngân sách dùng dữ liệu sản phẩm thật.
- Tra cứu đơn hàng theo mã đơn hoặc số điện thoại.
- Footer, dịch vụ bảo hành, floating chat.
- Vẫn đồng bộ dữ liệu: Admin sửa sản phẩm, web bán hàng reload là thấy.
