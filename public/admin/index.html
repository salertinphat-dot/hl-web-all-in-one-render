Hệ thống

Tổng hợp các chức năng vận hành, sao lưu dữ liệu và ghi chú kỹ thuật khi nâng cấp lên Backend thật.

⚙️

Ghi chú các chức năng cần Backend

1. Đăng nhập quản trị: Bản hiện tại đăng nhập dễ bằng HTML, không cần Backend JWT.

2. Middleware phân quyền ADMIN / STAFF: Khi nâng cấp Backend thật thì xử lý phân quyền ở server.

3. Upload hình sản phẩm: HTML gửi FormData. Backend có thể lưu local hoặc Cloudinary.

4. Lịch sử nhập / xuất kho: HTML đã có trang nhập/xuất kho và bảng lịch sử.

5. In hóa đơn bán hàng: HTML đã có nút in hóa đơn, mục hóa đơn bán lẻ riêng và tạo QR thanh toán VietQR và ghi nhận thanh toán thẻ.

6. Xuất Excel doanh thu: HTML đã xuất .xlsx bằng SheetJS khi demo offline; nếu Backend thật thì gọi API xuất Excel.

7. Backup database tự động: phải đặt ở Backend/Server, không nên để trong HTML.

8. Validate bằng Zod/Joi: phải đặt ở Backend. HTML có validate cơ bản bằng required/type.

9. Chống xóa sản phẩm đã phát sinh đơn hàng: demo offline đã chặn theo orderItems; Backend thật phải kiểm tra trong database.

10. Bảng nhà cung cấp: HTML đã có trang quản lý nhà cung cấp và liên kết sản phẩm/kho.
